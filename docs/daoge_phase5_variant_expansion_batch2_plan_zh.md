# DAOGE Phase5 第二批变体深化计划

## 目标

在第一批已经给 6 个主链补出首批变体入口之后，继续对 4 个最值得纵深的重点类做第二批变体深化：

1. `academic-figure-board`
2. `brand-packaging-board`
3. `map-route-board`
4. `illustrated-scene-set`

## 为什么只做这 4 类

这 4 类都满足三个条件：

1. 主链价值高
2. 变体差异清楚
3. catalog 入口密度还不够

相比之下，其他类目前先不急着继续补，是为了避免 catalog 过早膨胀成“数量很多但层级不清”的状态。

## 第二批建议落地的 4 个变体

### 1. academic-figure-board

- `multi-condition-comparison`

理由：

- 和 `graphical-abstract`、`mechanism-diagram` 都不同
- 能明确代表“实验对比图”这条高频路径

### 2. brand-packaging-board

- `beverage-label-design`

理由：

- 和 `brand-identity-board`、`cosmetic-packaging` 相比，更强调标签和瓶身识别
- 能体现包装类在“零售标签”上的专门路径

### 3. map-route-board

- `illustrated-city-map`

理由：

- 和 `travel-route-map`、`store-distribution-map` 相比，更偏导览和城市感受
- 能体现地图类不只服务功能路线，也能服务城市体验表达

### 4. illustrated-scene-set

- `concept-scene`

理由：

- 和 `healing-scene`、`picture-book-scene` 相比，更强调设定展示和世界观感受
- 能把插画类从“治愈/绘本”再推进到“概念设定”路径

## 本阶段修改内容

1. 新增 4 个变体 example
2. 更新 `examples.catalog.json`
3. 更新 README 与 examples README
4. 更新 smoke：
   - list 输出包含这 4 个 id
   - html 输出包含这 4 个 id
   - 至少 1 个新变体能走完整 demo

## 完成标准

满足以下条件，本阶段才算完成：

1. 新增 4 个变体入口
2. catalog 数量继续增长
3. smoke 和统一入口全绿
4. 至少 1 个第二批变体能从 catalog 跑到预检面板
